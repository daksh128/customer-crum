# Customer Churn Prediction

## Overview
This project predicts whether a telecom customer will leave the service using Machine Learning.

## Features
- Data Cleaning
- Label Encoding
- SMOTE Balancing
- Random Forest Model
- Model Evaluation

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
Telco Customer Churn Dataset

## Run Project
```bash
python cust.py
